mysql安装文档

  • 2022-01-06

宝塔安装：

Centos安装命令：

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

阿里云防火墙开通端口：

查看当前已经开放的端口: firewall-cmd –list-all
设置要开放的端口: firewall-cmd –zone=public –add-port=端口/tcp –permanent
重新加载：firewall-cmd –reload
相关链接：https://blog.csdn.net/qq_36640713/article/details/106553833

Linux安装MySQL步骤：

下载MySQL5.6

wget https://dev.mysql.com/get/Downloads/MySQL-5.6/MySQL-5.6.45-1.el7.x86_64.rpm-bundle.tar
wget https://downloads.mysql.com/archives/get/p/23/file/MySQL-5.6.51-1.el7.x86_64.rpm-bundle.tar
wget https://downloads.mysql.com/archives/get/p/23/file/mysql-8.0.22-1.el8.x86_64.rpm-bundle.tar

网址：https://downloads.mysql.com/archives/community/
步骤：MySQL官网(https://dev.mysql.com)->下载->社区服务器->档案->红帽企业版(RPM Bundle)

解压

tar -xvf MySQL-5.6.45-1.el7.x86_64.rpm-bundle.tar

查看之前使用rpm安装过的mysql

rpm -qa | grep -i mysql

删除之前使用安装包手动安装的mysql

使用whereis mysql查看mysq安装位置，将相关文件删除
使用locate mysql再次确认删除情况
输入 mysql -V确认mysql已经删除
(安装过程中，发现了残留的mysql配置文件, 删除)

删除

rpm -e
MySQL-server
MySQL-client
MySQL-devel

如果报依赖错误：
rmp -ev mysql-community-client-5.7.32-1.el7.x86_64 –nodeps
其中，mysql-community-client-5.7.32-1.el7.x86_64是rpm -qa找到的mysql包
Ctrl + R:查询历史记录

安装

rpm -ivh MySQL-client-5.6.45-1.el7.x86_64.rpm –force –nodeps
rpm -ivh MySQL-devel-5.6.45-1.el7.x86_64.rpm –force –nodeps
rpm -ivh MySQL-server-5.6.45-1.el7.x86_64.rpm –force –nodeps

切换到/usr/bin目录，启动

service mysqld start

停止mysql服务：

service mysqld stop
service mysql restart

获得密码

cd ~
ls -al
cat /root/.mysql_secret

登录

mysql -uroot -p’上一步获得的密码’

修改密码

SET PASSWORD FOR ‘root‘@’localhost’ = PASSWORD(‘root’);

登录控制

mysql -uroot -p123456
mysql>use mysql
mysql>select user,host,password from user;
mysql>delete from user where password=’’;
mysql>update user set host=’%’ where host=’localhost’;(%为所有，自己设置ip)
mysql> flush privileges

使用rpm安装mysql的默认目录：

1、数据库目录
/var/lib/mysql/
2、配置文件
/usr/share/mysql(mysql.server命令及配置文件) support-files=/etc/my.cnf
3、相关命令
/usr/bin(mysqladmin mysqldump等命令)
4、启动脚本
/etc/rc.d/init.d/(启动脚本文件mysql的目录)
systemctl restart mysqld.service
systemctl status mysqld
